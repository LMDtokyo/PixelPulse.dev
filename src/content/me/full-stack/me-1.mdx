### Backend: REST API на Spring Boot 3

Тонкий контроллер, бизнес-логика в сервисе, авторизация по ролям через `@PreAuthorize`:

```java
@RestController
@RequestMapping("/api/v1/listings")
@RequiredArgsConstructor
public class ListingController {

    private final ListingService service;

    @GetMapping
    public Page<ListingDto> active(@PageableDefault(size = 20) Pageable page) {
        return service.findActive(page);
    }

    @PostMapping
    @PreAuthorize("hasRole('SELLER')")
    public ResponseEntity<ListingDto> create(@Valid @RequestBody CreateListing req,
                                             @AuthenticationPrincipal AppUser user) {
        ListingDto created = service.create(req, user.getId());
        return ResponseEntity.status(HttpStatus.CREATED).body(created);
    }
}
```

### Аутентификация: JWT + Spring Security

Stateless-сессии, фильтр валидации JWT перед стандартным фильтром аутентификации:

```java
@Configuration
@EnableWebSecurity
@EnableMethodSecurity
@RequiredArgsConstructor
public class SecurityConfig {

    private final JwtAuthFilter jwtFilter;

    @Bean
    SecurityFilterChain chain(HttpSecurity http) throws Exception {
        return http
            .csrf(AbstractHttpConfigurer::disable)
            .sessionManagement(s -> s.sessionCreationPolicy(SessionCreationPolicy.STATELESS))
            .authorizeHttpRequests(reg -> reg
                .requestMatchers("/api/auth/**", "/actuator/health").permitAll()
                .anyRequest().authenticated())
            .addFilterBefore(jwtFilter, UsernamePasswordAuthenticationFilter.class)
            .build();
    }
}
```

### Real-time: WebSocket / STOMP

Брокер для пуша обновлений рынка и чата в реальном времени:

```java
@Configuration
@EnableWebSocketMessageBroker
public class WebSocketConfig implements WebSocketMessageBrokerConfigurer {

    @Override
    public void configureMessageBroker(MessageBrokerRegistry registry) {
        registry.enableSimpleBroker("/topic", "/queue");
        registry.setApplicationDestinationPrefixes("/app");
    }

    @Override
    public void registerStompEndpoints(StompEndpointRegistry registry) {
        registry.addEndpoint("/ws").setAllowedOriginPatterns("*").withSockJS();
    }
}
```

### Данные: PostgreSQL + JPA + Flyway

Сущность и версионированная миграция (схема в Git, не «руками в проде»):

```java
@Entity
@Table(name = "market_listing")
public class MarketListing {

    @Id
    @GeneratedValue(strategy = GenerationType.UUID)
    private UUID id;

    @ManyToOne(fetch = FetchType.LAZY)
    private AppUser seller;

    @Column(nullable = false)
    private String title;

    @Column(nullable = false)
    private BigDecimal price;

    @Enumerated(EnumType.STRING)
    private ListingStatus status = ListingStatus.ACTIVE;

    @CreationTimestamp
    private Instant createdAt;
}
```

```sql
-- V12__market_listing.sql
create table market_listing (
    id         uuid primary key default gen_random_uuid(),
    seller_id  uuid not null references app_user (id),
    title      text not null,
    price      numeric(12, 2) not null check (price >= 0),
    status     text not null default 'ACTIVE',
    created_at timestamptz not null default now()
);

create index idx_listing_status on market_listing (status);
```

### Кэш: Redis

Горячие выборки кэшируются, инвалидация — при записи:

```java
@Service
@RequiredArgsConstructor
public class ListingService {

    private final ListingRepository repository;

    @Cacheable(value = "listings:active", key = "#page.pageNumber")
    public Page<ListingDto> findActive(Pageable page) {
        return repository.findByStatus(ListingStatus.ACTIVE, page).map(ListingDto::from);
    }

    @CacheEvict(value = "listings:active", allEntries = true)
    public ListingDto create(CreateListing req, UUID sellerId) {
        var saved = repository.save(MarketListing.from(req, sellerId));
        return ListingDto.from(saved);
    }
}
```

'use client'
import { useQuery } from '@tanstack/react-query'

export function ListingGrid({ initial }: { initial: Listing[] }) {
  const { data } = useQuery({

### BFF-прокси на Fastify

Тонкий слой между фронтом и Java-бэкендом: проксирует токен, скрывает внутренние эндпоинты:

```typescript
import Fastify from 'fastify'

const app = Fastify({ logger: true })

app.get('/bff/listings', async (req, reply) => {
  const upstream = await fetch(`${process.env.API_URL}/api/v1/listings?status=ACTIVE`, {
    headers: { authorization: req.headers.authorization ?? '' },
  })
  reply.code(upstream.status).send(await upstream.json())
})

await app.listen({ port: 8080, host: '0.0.0.0' })
```

### Боты: Discord.py + Steam API

Async-бот LunaMarket: отдаёт цены и статус предметов прямо в Discord:

```python
import aiohttp
import discord
from discord.ext import commands

bot = commands.Bot(command_prefix="!", intents=discord.Intents.all())

@bot.command(name="price")
async def price(ctx: commands.Context, appid: int):
    url = f"https://store.steampowered.com/api/appdetails?appids={appid}&cc=us"
    async with aiohttp.ClientSession() as session, session.get(url) as resp:
        payload = await resp.json()

    data = payload[str(appid)]["data"]
    price = data.get("price_overview", {}).get("final_formatted", "free")
    await ctx.send(f"**{data['name']}** — {price}")

bot.run(TOKEN)
```

### Security: реверс-инжиниринг и анализ

Защитный RE — разбираю образцы (малварь, читы), чтобы вытащить IOC и построить детект, не запуская их:

```bash
diec sample.exe                              # идентификация упаковщика/протектора
dnspy.console.exe -o out/ sample.exe         # декомпиляция .NET-сборки
rabin2 -zz sample.bin | rg 'https?://|\.dll' # строки → C2-домены, импорты
# далее — Ghidra/IDA для нативного кода, x64dbg для рантайм-дампа
```

## Текущий проект: LunaMarket

Игровая платформа: форум, торговая площадка и система достижений.

- **Интеграции:** Discord- и Steam-боты (async Python).
- **Инфраструктура:** Docker Compose, Nginx, метрики через Prometheus + Micrometer.
