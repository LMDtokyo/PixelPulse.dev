## Примеры работ и методик

### Сетевой сканер на C++/Rust

**Задача:** Создать утилиту для анализа сетевого трафика, обнаружения открытых портов и сервисов.
**Инструменты:** `libpcap` для C++, `tokio` для Rust.

#### Пример на C++:
```cpp
#include <iostream>
#include <pcap.h>

int main() {
    char errbuf[PCAP_ERRBUF_SIZE];
    pcap_if_t *alldevs;
    if (pcap_findalldevs(&alldevs, errbuf) == -1) {
        std::cerr << "Error finding devices: " << errbuf << std::endl;
        return 1;
    }

    // Выбор интерфейса, установка фильтра (например, "tcp port 80"), захват пакетов
    // Анализ трафика для определения открытых портов и сервисов

    pcap_freealldevs(alldevs);
    return 0;
}
```
Пример на Rust (с использованием async-экосистемы):
```cpp
use tokio::net::TcpStream;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let addr = "192.168.1.10:80";
    match TcpStream::connect(addr).await {
        Ok(_) => println!("Port 80 is open on {}", addr),
        Err(e) => println!("Port 80 closed or filtered on {}: {}", addr, e),
    }
    Ok(())
}

```
Консольный вывод (пример):
```cpp
$ ./scanner --host 192.168.1.10 --ports 1-1000
[INFO] Scanning host 192.168.1.10...
[OPEN] Port 80: HTTP Service
[CLOSED] Port 443: No Response
[INFO] Scan complete.

```

Автоматизация и OSINT на Python
Задача: Быстро собрать данные о целевых хостах или публичных профилях для OSINT.
Инструменты: requests, BeautifulSoup для парсинга.
```cpp
import requests
from bs4 import BeautifulSoup

url = "https://example.com/users"
r = requests.get(url)
soup = BeautifulSoup(r.text, 'html.parser')
profiles = soup.find_all("div", class_="user-card")

for p in profiles:
    name = p.find("h2").text.strip()
    email = p.find("a", class_="email").text.strip()
    print(f"[INFO] Found user: {name}, email: {email}")
```
Вывод:
```cpp
[INFO] Found user: Alice Johnson, email: alice@example.com
[INFO] Found user: Bob Smith, email: bob@example.org

```
